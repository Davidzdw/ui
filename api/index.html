Button

A circular button with optional aftertouch.
Basic: 
var dial = new mt.dial('#dial')
Advanced: 
var dial = new mt.dial('#dial',{
    mode: 'toggle',
    state: true,
    size: [100,100],
    event: (v) => {
      alert(v);
    }
  })

Options

        
  size: [80,80]
  event: console.log
mode
string
Interaction mode. Options are: aftertouch, toggle, button, impulse 
dial.mode = 'toggle'
state
boolean
Whether the button is pressed or not
dial.state = true
turnOn()
Press the button
dial.turnOn()
turnOff()
Unpress the button
dial.turnOff()
flip( state )
Toggle the button to its opposite state
dial.flip()
